  • globalTurbulencePolygonsUpToEnabled

    DataTypeOptions.turbulencePolygons are generated for specific history time set inSkyPath.dataHistoryTime. Polygons for smaller history time will not be available while offline and need a new fetch after changing SkyPath.dataHistoryTime. For example if SkyPath.dataHistoryTime is set to 4h, DataTypeOptions.turbulencePolygons will be available for 4h offline, but not available for 2h, 1h and 0.5h. Set this flag to allow fetching polygons generated for each history time up to set in SkyPath.dataHistoryTime to have it available offline. This will increase network traffic Default is false.

    Declaration

    Swift

    public var globalTurbulencePolygonsUpToEnabled: Bool
  • altRange

    Used only for data that can’t be filtered by altitude locally and need to make a server request. For example, turbulence polygons GeoJSON. In thousands of feet. Max range is 0…50. Optional.
