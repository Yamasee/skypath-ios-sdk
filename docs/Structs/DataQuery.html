    Polygon coordinates to get turbulence inside it. Should have max 250 coordinates, otherwise SkyPath.didFailToFetchNewData will be called with an error. Simplify polygon with enough tolerance.

    Visible polygon coordinates to get turbulence inside it. Should have max 250 coordinates, otherwise SkyPath.didFailToFetchNewData will be called with an error. Simplify polygon with enough tolerance.

    Swift

    public var viewport: [CLLocationCoordinate2D]?

    Wheather to receive global data like turbulence polygons, global traffic and pireps. When false and the polygon is not nill the data inside polygon will be fetched only. Default is true.

    Swift
