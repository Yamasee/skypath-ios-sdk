AlertQuery

public struct AlertQuery

Control how to search alerts. By default it uses current location and beam mode with default values. When pass route the search will be done in coordinates instead. By default, the current location altitude will be used to make a range with +/- 2k feet (it’s dynamic from the server so can be different). Set altRange when you need custom range to search for alerts.
