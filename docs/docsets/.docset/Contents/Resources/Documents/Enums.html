  • DataUpdateFrequency

    Determines the time interval to fetch data from the server. If for example network traffic should be kept to minimal and it’s ok to receive data more rarely set minimal. The fast will try to update data as soon as possible. When you would like to stop receiving data (for example when SkyPath layer is hidden in the app) set it to none.
