TurbulenceResultType

public enum TurbulenceResultType

SDK stores own tracked turbulence for some time locally on device and also receives turbulence from the server reported by other devices. Use it in TurbulenceQuery.

  • server

    Turbulence from the server, aggregated based on multiple reports and per current airplane type.

  • serverExcludeOwn(timeSpan:)

    Use it if you show own and server tiles differently. This will return server turbulence without turbulence for own tiles that a/c has passed for last timeSpan. Turbulence for tiles that has been passed more than timeSpan from now will be included in the result.
