  • dataUpdateFrequency

    Configure the frequency of data update from the server. For example, to save network traffic, you can set it to none to disable periodic new data fetch from the server at all, or set to minimal to update with bigger time intervals. Can be updated at any time in the project. Value is stored across app launches until changed. Default is fast.

    Declaration

    Swift

    public var dataUpdateFrequency: DataUpdateFrequency { get set }
  • dataHistoryTime

    Set history time to receive data for in an initial data request made on start. When started, SDK fetches initial data for time back in time and then receive only updates since last update. So if you don’t need for example 6 hours of data history, set this variable to number of hours that best suites your needs and save network traffic. Default is twoHours.
